# starter-ts-api

Minimal Node.js + TypeScript API starter project.

## Requirements

- Node.js 20+

## Quick start

```bash
cd starter-ts-api
npm install
npm run dev
```

Server URL: `http://localhost:3000`

## Scripts

- `npm run dev`: run server in watch mode
- `npm run build`: compile TypeScript to `dist/`
- `npm run start`: run compiled app from `dist/`

## Endpoints

- `GET /`
- `GET /health`
